# CSE360 Homework 4 - Joshua Wright

## About

This repository contains the source code, tests, and documentation for CSE360 Homework 4 (Fall 2025). The project implements the Staff role functionality for the Student Question and Answer System, including CRUD operations, internal documentation, JUnit testing, and a demonstration screencast. Key goals include:

* Implementing the Staff role user stories
* Reviewing questions, answers, and private feedback
* Flagging content and tracking resolutions
* Adding private notes for instructors and staff
* Generating activity reports
* Maintaining consistent internal and Javadoc documentation
* Automated testing using JUnit
* Screencast demonstration of implemented features
